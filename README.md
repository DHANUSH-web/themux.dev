# themux.dev

The official website for [mux](https://themux.dev) — a fast, lightweight, and open-source C/C++ project manager built in Rust.

## Stack

- [Next.js 16](https://nextjs.org) with App Router
- [React 19](https://react.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- TypeScript

## Development

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Install Scripts

The site serves install scripts for the mux CLI:

- **Linux/macOS:** `curl -fsSL https://themux.dev/install.sh | sh`
- **Windows:** `irm https://themux.dev/install.ps1 | iex`
